# react 时间范围选择器

基于 React 的移动端范围选择器。

该组件可以通过拖拽修改一定范围内的跨度。

## 特性

## 安装

```bash
npm install react-time-range-selector
```

或者

```bash
yarn add react-time-range-selector
```
